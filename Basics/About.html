C is one of the most popular computer languages even today because it is structured, high-level, machine independent language.
  The root of all modern languages is ALGOL, introduced in the early 1960s. ALGOL was the first computer language to use a block structure. ALGOL gave the concept of structured programming to the computer science community. In 1967, Martin Richards developed a language called BCPL (Before Combined Programming Language) primarily for writing system software.
  C was evolved from ALGOL, BCPL and one B by Dennis Ritchie at the Bell Laboratories in 1972. Since, it was developed along with the UNIX operating system, it is strongly associated with UNIX

UNIX is one the most popular network operating system in use today and the heart of the Internet data superhighway.

During the 1990's, C++, a language completely based on C, underwent a number of improvements and changes and became an ANSI/ISO approved language in November 1977. C++ added several new features to C to make it not only a true Object - Oriented language but also a more versatile language. During the same period, Sun Microsystems of USA created a new language Java modelled on C and C++.
