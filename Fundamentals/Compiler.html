Until now, we have seen what a Computer is, how it works and what language it speaks rather understands, we also understood the language levels a computer can have as an input. So, the main objective of this section remains to be simple and that is; how a computer converts a Source Code written in High Level language to Machine Level language.

Terms Every Programmer Should Know :

1. Source Code : In computing, source code is any collection of code, with or without comments, written using a human-readable programming language, usually as plain text.
2. Compiler : A program that converts instructions into a machine-code or lower-level form so that it can be read and executed by a computer.
2. Executable Program : In computing, executable code, an executable file, or an executable program, sometimes simply referred to as an executable or binary, causes a computer "to perform indicated tasks according to encoded instructions", as opposed to a data file that must be interpreted (parsed) by a program to be meaningful.

Compiling a Source Code :

- As we have read, program written in High Level Language  is called the Source Code.
- A program called "Compiler" is used to translate a source  program into a Machine Level code also termed as an
  Object Program.

☢ Compiler also checks for syntactic and semantic errors.

- The Object Program is often then linked with other supporting library code before the object can be executed on the machine.

Explanation :

Object File which is initially present in Machine Language often has two parts; one is the Operand part and the other is the Operator. Linker searches in the Library Code and finds out what the operator means to do with the operand. It then links the operand with the operator and then the object file is converted into an Executable File

☢ Executable File → C/C++ : .exe(Windows), .out(Linux),
  .app(MacOS), Java : .class / .jar file
☢ Python and javascript on the contrary are interpreted
 languages, they don't have any compiler rather they have
 their own interpreter.

Example : (Source Code)

printf("Hello World");


printf → Operator written in the library code.
"Hello World" → Operand

Example : (Machine Code)

0x48,0x65,0x6c,0x6c,0x6f,0x20,0x57,0x6f,0x72,0x6c,
0x64,0x0a;
