In this tutorial, we will be learning about the C programming language.It is a middle-level language, so it is easy to learn. Moreover, it is a general-purpose language, which can be used for many different purposes and helps beginners to understand the fundamentals of computer theories.

Advantages of C programming language :

1. Easy to understand
2. Many libraries present
3. Fast Execution (fast run-time)
4. Easy to debug.
5. Procedure Oriented Programming
6. Dynamic Memory Allocation

Disadvantages of C programming language :

1. Absence of Object Orientation
2. Inefficient memory management
3. Absence of garbage collection
4. Absence of exception handling
