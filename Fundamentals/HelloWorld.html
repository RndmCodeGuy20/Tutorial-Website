Programming :

The process of creation of an ordered set of instructions to solve a problem with the use of a Computer.

☢ The most important part is to perfectly and efficiently     arrange these instructions.

Programming Fundamentals :

Computer programming (often shortened to programming) is a process that leads from an original formulation of a computing problem to executable computer programs. Simply put, it is the process of designing and building an executable computer program to accomplish a specific computing result or to perform a specific task. Programming involves activities such as analysis, develop understanding, generating algorithms, verifying requirements of algorithms including their correctness and resource consumption, and finally implementation of algorithms in a target programming language.

Programming Languages :

A programming language is a formal language comprising a set of strings that produce various kinds of machine code output. Programming languages are one kind of computer language, and are used in computer programming to implement algorithms.

How to start programming :

There are multiple environments to write your code and execute it :-

1. A Text Editor / IDE, this is an application that allows you to write your code in a text file. Text Editors provide a rich set of features to write your code. Some of the features are :-

- Syntax Highlighting : This feature highlights the syntax of   your code.
- Auto-completion : This feature automatically completes     the code you are writing.
- Auto-indentation : This feature automatically indents your   code.
- Auto Save : This feature automatically saves your code.
- Integrated Terminal : This feature allows you to run your     code in a terminal.

2. A Terminal, this is an application that allows you to run your code in a terminal. Command Line Interface Editors (CLI Eds) like Vim, EMacs, Gedit, Nano ,etc. are examples of CLI based Editors. They are used to simply write your code and execute it, some of them support syntax highlighting.

3. A Web-based Text Editor, this is an application that allows you to run your code in a web browser on specific web platforms like Codepen, Repl.it. Web Browsers like Chrome, Firefox and Safari are examples of web browsers. These editors support linting (syntax highlighting), auto completion, auto indentation, auto save and integrated terminal, and does not require any installation.
☢ Best for beginners.

You can use any of these environments to write your code and execute it. But for the first few lectures, we will be using the Web-based Text Editor (Repl.it).

How to write code :

1. As we are using a web based code editor we will first be redirecting ourselves to the site, in our case Repl.it.

2. Sign / Log - in to the site and create a repl, select C language in the templates drop down and give an appropriate name to your project.

3. After creating a repl, you will be redirected to a window with a file explorer tab, editor tab and a terminal tab.

4. We will be writing our code in the editor area and after running our code the result will be visible in the terminal area.


☢ Write this piece of code into the editor area :-

Understanding the Code :

- The first line of the code is the #include <stdio.h>. This is a pre-processor directive. It is used to include the header file <stdio.h> which stands for standard Input-Output header file (I/O). Basic I/O commands like printf(); and scanf(); are defined in this header file.

☢ C/C++ is a case-sensitive language. So, printf(); is not the same as Printf();. Also, it is not the same as PRINTF();.

- The second line of the code is the int main();. This is the main(); function, it is the function that is executed first when the program execution starts. This is the starting point of any C/C++ program.

- These parentheses {} are the start and end of the code block, the code to be executed is written inside these parenthesis.

- The fourth line is the printf("Hello,World");. This is the printf(); function, it is used to print the string "Hello, World!" in the output area.
