This section focuses on the principles on which a computer internally works and the principles of the modern digital computers.

The computer does two things and two things only :-

- Store the Data
- Manipulate the Data,

and it is very good at these two things.

Computerization : The impact, role and use of a computer to do the things previously done by humans or by machines.

The Computer :
A device made up of the following three basic elements, namely :-

1. Input Device : It reads into the Central Processing Unit.

2. Central Processing Unit : The element of the computer consists of the following three units wired together. They are :

2.A Main Storage Unit : The incoming data of the input unit and results of the processing are held in this unit. This unit consists of millions of continuous memory locations or cells, each of which is capable of holding one or more characters.
2.B Arithmetic and Logic Unit (A.L.U.) : It performs all the arithmetic and logical operations on the data and the instructions.
2.C Control Unit : It issues the signals to the other units for performing the required operations. Instructions and data held in the main storage are brought into this unit where they are interpreted and passed on to an appropriate unit.

3. Output Device : After performing the required operations, each unit sends back the result to the Main Storage. These results are then sent to the requested output device which converts them into human readable form.

Historical Development of Digital Computers :

In 1642, Blaise Pascal constructed a mechanical calculator. This machine was composed of wheels or dials each marked from 0 to 9. This could add and subtract, multiplication and division was done through repeated addition and subtraction. Later, Leibnitz in 1671 improved upon Pascal's machine which could only add and subtract. Leibnitz's machine could perform all the four basic functions of arithmetic.

Charles Babbage constructed an Analytical Engine. The idea embodied in this device was that once the instructions and data are fed into the machine via punched cards, it would work upon it without the intervention of the operator.


Electronic Calculator Versus Computer : The two devices perform the same arithmetic operations of addition, subtractions, multiplication and division. The electronic calculator requires a man on its side to provide it the instructions by depressing its specific keys every now and then, whereas, the digital computer once supplied with the instructions to be executed and the data to be operated on, goes on automatically.

Process Execution :

1. "Variables, data, executable programs, etc." are first stored in the "Main Memory", through input devices or "On-board storage devices" (SSD, HDD, CD-ROM).

2. Data in the "Main Memory" is merged/sorted in different addresses/registers as per the users demand.

3. The "Control Unit" then fetches the stored data and program from the "Main Memory".

4. The "Control Unit" then analyzes the data that it has fetched, the process of analyzing the data is also known as decoding.

5. After the data is analyzed (decoded) by the "Control Unit", it is then forwarded to the "Arithmetic and Logic Unit (A.L.U.)".

6. All the necessary Arithmetic or logical operations are done here in the "A.L.U.", also known as execution.

7. After execution of the required program the result (either final or intermediate) is copied back to the "Main Memory".

Summary :

FETCH   ⟶   DECODE   ⟶   EXECUTE   ⟶   STORE
